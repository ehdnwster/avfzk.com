サイトマップについて

本資料では、AVFZK における
サイトマップの運用方法について説明します。

--------------------------------------------------

サイトマップ URL
https://www.avfzk.com/sitemap.xml

--------------------------------------------------

生成方法

- PHP スクリプトによる自動生成
- 公開ページのみを対象
- 重複 URL の除外

--------------------------------------------------

設定内容

- 更新頻度：weekly
- 優先度：主要ページは高めに設定
- 深すぎる階層は含めない

--------------------------------------------------

対応検索エンジン

- Google Search Console
- Bing Webmaster Tools
- Yandex Webmaster

--------------------------------------------------

運用方針

- データ更新時に再生成
- 定期的なインデックス確認
