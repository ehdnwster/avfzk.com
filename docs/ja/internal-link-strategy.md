# 内部リンク戦略（Internal Link Strategy）

このページでは、AVFZK サイトにおける **内部リンク設計** の考え方と実装方法を解説します。

対象サイト  
https://www.avfzk.com/

---

## 1. 内部リンクの目的

- サイト内の回遊性向上
- 関連ページの関連性を検索エンジンに伝える
- 重複コンテンツ回避
- 特定ジャンルや女優ページへの自然誘導

---

## 2. 主な内部リンク構造

### 2.1 ナビゲーション（ヘッダー / メニュー）

- ホーム `/`
- 新着情報 `/`
- 本日出勤 `/today/`
- 女優一覧 `/girllist/`
- エリア `/area/`
- ジャンル `/genre/`
- 無修正 `/uncensoredlist/`
- 人気 `/popular/`

### 2.2 コンテンツ内リンク

- 女優ページ → 所属ジャンル・エリアページ
- ジャンルページ → 関連女優一覧
- エリアページ → 出勤中の女優一覧
- 一覧ページ → 個別女優ページ（Person Schema と連動）

---

## 3. SEO向けの内部リンクルール

1. **クローラーが辿れるリンクだけを設置**
2. **リンクテキストはキーワードを含む**  
   例: `AV女優一覧はこちら`
3. **外部リンクは別ドメインに限定**  
4. **パンくずリストは必ず設置**  
   例: `ホーム > 女優一覧 > 個別女優`
