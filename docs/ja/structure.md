URL 構造

設計方針

- シンプルで理解しやすい URL
- 階層の浅い構造
- 内部リンクによる関連付け

--------------------------------------------------

内部ナビゲーション

- パンくずリストの導入
- 女優ページからジャンル・エリアへのリンク
- 一覧ページから詳細ページへの導線

--------------------------------------------------

検索エンジン対応

- クローラが巡回しやすい構成
- 不要なパラメータ URL を排除
- 正規 URL（canonical）の指定
