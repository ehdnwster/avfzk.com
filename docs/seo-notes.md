SEO Design Notes

This document summarizes SEO-related design decisions
used throughout the AVFZK website.

--------------------------------------------------

SEO Goals

- Clear content categorization
- Indexable internal structure
- Fast discovery of new pages
- High compatibility with search engines

--------------------------------------------------

Content Strategy

- Pages focus on informational value
- Actress profiles use unique descriptions
- Category pages act as navigation hubs

--------------------------------------------------

Internal Linking Strategy

- Actress profile pages link to:
  - Related genre pages
  - Area index pages
- Category pages link back to profiles
- Breadcrumb navigation reinforces hierarchy

--------------------------------------------------

Metadata & Markup

- Unique title and meta description per page
- JSON-LD structured data embedded
- Canonical URLs used to avoid duplication

--------------------------------------------------

Content Policy

- No hosted video files
- No direct downloads
- Informational listings only

--------------------------------------------------

Long-Term Optimization

- Continuous content updates
- Sitemap regeneration on data changes
- Index coverage monitoring via webmaster tools
